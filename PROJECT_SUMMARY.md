## 📞 技术支持

### 常见问题

**Q: 为什么看不到 `/help` 中没有显示 `/new`？**
```
A: 正常的，因为 hidden: true，这是有意设计的
   来保持菜单的整洁和专业性。
   /new 命令仍然可以直接使用。
```

**Q: `/new` 和 `/session new` 有什么区别？**
```
A: 两者功能完全相同，/new 只是快捷方式。
   命令执行的逻辑完全相同，结果也完全相同。
```

**Q: 可以修改隐藏状态吗？**
```
A: 可以，编辑 newCommand.ts 中的 hidden 字段：
   • hidden: true  - 隐藏（默认）
   • hidden: false - 显示在菜单中
```

**Q: 如何添加更多快捷命令？**
```
A: 按照 newCommand.ts 的模式创建新文件，然后在
   BuiltinCommandLoader.ts 中导入并注册即可。
```
