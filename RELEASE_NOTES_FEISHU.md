# Easy Code × 飞书：你的 AI 办公助手来了

## 一句话

现在你可以直接用自然语言告诉 Easy Code，让它帮你在飞书里完成日常办公操作——不用打开飞书，不用找入口，说一句就行。

---

## 能做什么

### 📅 日程管理

不用切到飞书日历翻来翻去，直接问 AI。

- "帮我看下今天的会议安排"
- "今天下午 3 点到 4 点有没有空？"
- "帮我创建一个明天上午 10 点的会议，标题是'产品评审'"
- "帮我找一下下周可用的会议室"

### 📝 飞书文档

一句话创建文档，内容、排版都帮你搞定。

- "帮我创建一个飞书文档，标题是'Q3 季度复盘'，内容写一份复盘模板"
- "把这段文字整理成飞书文档"
- "帮我搜索飞书里包含'设计规范'的文档"

### 📊 飞书表格

创建表格、读写数据，不用手动操作。

- "帮我创建一个飞书表格，标题是'本周迭代任务'"
- "帮我往表格里写入这些数据"
- "帮我在飞书表格里查找包含'紧急'的行"

### 💬 飞书消息

发消息、查聊天，不用切窗口。

- "帮我给研发群发一条消息：今天下午 3 点开会"
- "帮我搜索和'张伟'的聊天记录里提到'服务器'的消息"
- "帮我看一下最近有哪些群聊"

### ☁️ 云文档管理

上传、下载、搜索云盘文件。

- "帮我上传这个文件到飞书云盘"
- "帮我在飞书云盘搜索'合同'相关的文件"
- "帮我把这个飞书文档导出成 PDF"

### ✅ 飞书任务

创建和追踪待办事项。

- "帮我创建一个飞书任务：下周一前完成接口文档"
- "帮我看一下我的待办任务"
- "把任务'提交代码'标记为完成"

### 📧 飞书邮件

读邮件、发邮件，不用切到邮箱。

- "帮我给 xxx@company.com 发一封邮件，主题是'会议确认'"
- "帮我看看收件箱里最新的邮件"
- "帮我回复这封邮件：好的，收到"

### 📚 知识库

查找和创建知识库页面。

- "帮我看一下有哪些知识库"
- "在产品知识库里创建一个新页面，标题是'新手指南'"
- "帮我查一下知识库里的'部署流程'页面"

### 🎥 会议与妙记

查看会议记录、回放。

- "帮我查一下昨天的会议记录"
- "帮我搜索妙记里提到'OKR'的内容"

### 🎯 OKR

查看和更新目标进度。

- "帮我看看当前的 OKR 周期"
- "帮我更新一下'提升系统稳定性'这个目标的进展"

### 👤 通讯录

查人、找联系方式。

- "帮我搜索一下'李明'的飞书信息"

---

## 怎么用

1. **首次使用**：直接说你想做什么（比如"帮我看飞书日历"）。如果还没有授权，Easy Code 会自动弹出一个链接，在浏览器里扫码登录即可，全程自动引导。
2. **之后使用**：直接说就行，不用再登录。

> 💡 **小技巧**：不确定某个功能怎么用？直接问就行，比如"飞书能帮我发邮件吗？" Easy Code 会告诉你具体怎么操作。

## 支持的飞书功能一览

| 类别 | 支持的操作 |

| 日历 | 查看日程、创建会议、查询空闲时间、找会议室、回复邀请、智能推荐时间 |
| 文档 | 创建文档、获取内容、更新文档、搜索文档、插入图片 |
| 表格 | 创建表格、读写单元格、追加行、查找、导出 |
| 消息 | 发消息、查看聊天、搜索消息、回复消息、创建群聊、置顶消息、下载附件 |
| 云盘 | 上传文件、下载文件、创建文件夹、搜索文件、导出文档 |
| 任务 | 查看待办、创建任务、完成任务、搜索任务 |
| 邮件 | 发邮件、查看收件箱、读邮件、回复邮件、创建草稿 |
| 知识库 | 查看知识库、列出页面、创建页面、查看页面详情 |
| 会议/妙记 | 搜索会议、查看录制、查看会议纪要、搜索妙记 |
| OKR | 查看周期、查看详情、更新进展 |
| 通讯录 | 搜索同事、查看信息 |
| 演示文稿 | 创建幻灯片 |
